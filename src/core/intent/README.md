# Intent

## Responsabilidad

Representa lo que el usuario quiere conseguir.

## Entrada

- Lenguaje natural
- Voz
- Texto
- Eventos automáticos

## Salida

Una intención estructurada e independiente del proveedor de IA.

## Ejemplos

Usuario:

> "Pon música."

Intent:

play_music

---

Usuario:

> "¿Qué tengo hoy?"

Intent:

daily_briefing

---

Usuario:

> "Apaga el ordenador."

Intent:

shutdown_system
