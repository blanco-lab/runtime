# Runtime HQ

## ¿Qué es Runtime HQ?

Runtime HQ (Headquarters) es el centro de operaciones oficial del proyecto Runtime.

Toda decisión técnica, tarea, reunión, informe o planificación deberá registrarse aquí.

## Principios

1. El conocimiento pertenece al proyecto, no a las conversaciones.
2. Toda decisión importante debe quedar documentada.
3. Primero decidimos.
4. Después documentamos.
5. Después construimos.

## Organización

dashboard/
Estado actual del proyecto.

tasks/
Tareas abiertas, en curso y completadas.

meetings/
Actas de reuniones del equipo.

reports/
Informes técnicos.

decisions/
Architecture Decision Records (ADR) y decisiones relevantes.

sprints/
Planificación de Sprint.

agents/
Información de los agentes del equipo.
